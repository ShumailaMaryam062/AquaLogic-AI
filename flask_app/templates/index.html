💡 How AquaLogic AI Works

🧠

DFS Algorithm

Uses Depth-First Search to explore all possible states and find the optimal path to your goal.

🔧

Operations

  • Fill — Fill jug to max
  • Empty — Empty completely
  • Pour — Transfer between jugs
📊

State Space

Each state (x, y) represents water levels in both jugs. The algorithm tracks visited states to avoid cycles.

🎯

Goal Condition

Solution is found when either jug contains exactly the target amount of water.

📐 Mathematical Insight

A solution exists if and only if the target is divisible by GCD(jug1, jug2) and doesn't exceed the larger jug.

target mod GCD(jug₁, jug₂) = 0 ∧ target ≤ max(jug₁, jug₂)
